scalor:auto-conf

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.0.20171215014228:auto-conf

Description:

Discover and inject maven project properties for use by other executions. Generated properties look like: [scalor.autoconfSomeName]. Interpolated parameters look like: [scalor.othermoduleSomeName]. Interpolated parameters are collected from all scalor plugin executions.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: initialize.
