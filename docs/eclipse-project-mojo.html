scalor:eclipse-project

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.0.20171215014228:eclipse-project

Description:

Provision and configure Eclipse .project file.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
eclipseOptionSeparator String - Eclipse settings option list separator.
Default value is: [;\n].
User property is: scalor.eclipseOptionSeparator.
eclipseProjectBuildApply boolean - Enable create/delete for .project 'projectDescription/buildSpec' entries.
Default value is: true.
User property is: scalor.eclipseProjectBuildApply.
eclipseProjectBuildCreate String[] - Append these Eclipse builders into the .project 'projectDescription/buildSpec' section.
Default value is: org.scala-ide.sdt.core.scalabuilder,org.eclipse.m2e.core.maven2Builder.
User property is: scalor.eclipseProjectBuildCreate.
eclipseProjectBuildDelete String[] - Remove these Eclipse builders from the .project 'projectDescription/buildSpec' section.
Default value is: org.eclipse.jdt.core.javabuilder.
User property is: scalor.eclipseProjectBuildDelete.
eclipseProjectBuildOrdering String - Order Eclipse .project file entries according to these rules. Rule format: item = path, where: 'item' - relative sort order, 'path' - regular expression to match against 'projectDescription/buildSpec/buildCommand/name',
Default value is: 01 = org.scala-ide.sdt.core.scalabuilder ; 02 = org.eclipse.m2e.core.maven2Builder ; .
User property is: scalor.eclipseProjectBuildOrdering.
eclipseProjectBuildReorder boolean - Re-order Eclipse .project file by 'projectDescription/buildSpec/buildCommand/name' .
Default value is: true.
User property is: scalor.eclipseProjectBuildReorder.
eclipseProjectFile File - Location of Eclipse .project configuration file.
Default value is: ${project.basedir}/.project.
User property is: scalor.eclipseProjectFile.
eclipseProjectNameApply boolean - Enable enforcement for .project 'projectDescription/name'.
Default value is: true.
User property is: scalor.eclipseProjectNameApply.
eclipseProjectNamePattern String - Derive Eclipse project name from maven variables.
Default value is: ${project.artifactId}.
User property is: scalor.eclipseProjectNamePattern.
eclipseProjectNatureApply boolean - Enable create/delete for .project 'projectDescription/natures/nature' entries.
Default value is: true.
User property is: scalor.eclipseProjectNatureApply.
eclipseProjectNatureCreate String[] - Append these Eclipse natures into .project 'projectDescription/natures' section.
Default value is: org.eclipse.jdt.core.javanature,org.scala-ide.sdt.core.scalanature,org.eclipse.m2e.core.maven2Nature.
User property is: scalor.eclipseProjectNatureCreate.
eclipseProjectNatureDelete String[] - Remove these Eclipse natures from .project 'projectDescription/natures' section.
User property is: scalor.eclipseProjectNatureDelete.
eclipseProjectProcess boolean - Enable processing of Eclipse .project file.
Default value is: true.
User property is: scalor.eclipseProjectProcess.
eclipseProjectProvide boolean - Provide Eclipse .project file when missing.
Default value is: true.
User property is: scalor.eclipseProjectProvide.
eclipseProjectTemplate String - Scalor plugin classpath resource providing Eclipse .project file template.
Default value is: META-INF/template/.project.
User property is: scalor.eclipseProjectTemplate.
skip boolean - Skip all steps of all executions.
Default value is: false.
User property is: scalor.skip.
