scalor_2.12:prepack-linker-test

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.0.0.20180101211449:prepack-linker-test

Description:

Prepare linker build resources for packaging for compilation scope=test.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: process-test-classes.

Optional Parameters

Default value is: META-INF/resources/script.
User property is: scalor.linkerMetaFolder.

Default value is: ${project.build.directory}/scala-js/test.
User property is: scalor.linkerTestTargetFolder.

Default value is: true.
User property is: scalor.prepackEnsureOutput.
