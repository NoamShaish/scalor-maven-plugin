scalor:link-scala-js-test

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.0.1.20180112035848:link-scala-js-test

Description:

Generate Scala.js runtime JavaScript for scope=test.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-test-classes.
