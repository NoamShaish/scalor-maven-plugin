Plugin Documentation

Goals available for this plugin:

Goal Report? Description
scalor:clean-macro No Clean project cache resources for compilation scope=macro.
scalor:clean-main No Clean project cache resources for compilation scope=main.
scalor:clean-test No Clean project cache resources for compilation scope=test.
scalor:compile-macro No Compile Java and Scala sources in compilation scope=macro.
scalor:compile-main No Compile Java and Scala sources in compilation scope=main.
scalor:compile-test No Compile Java and Scala sources in compilation scope=test.
scalor:eclipse-config No Install companion Eclipse plugin provided by this Maven plugin when running under Eclipse/M2E.
scalor:link-scala-js-main No Generate Scala.js runtime JavaScript for scope=main.
scalor:link-scala-js-test No Generate Scala.js runtime JavaScript for scope=test.
scalor:prepack-linker-main No Prepare linker build resources for packaging for compilation scope=main.
scalor:prepack-linker-test No Prepare linker build resources for packaging for compilation scope=test.
scalor:prepack-macro No Prepare build resources for packaging for compilation scope=macro.
scalor:prepack-main No Prepare build resources for packaging for compilation scope=main.
scalor:prepack-test No Prepare build resources for packaging for compilation scope=test.
scalor:register-macro No Register Java, Scala, Resource folders for compilation scope=macro.
scalor:register-main No Register Java, Scala, Resource folders for compilation scope=main.
scalor:register-test No Register Java, Scala, Resource folders for compilation scope=test.
scalor:scaladoc-macro Yes Produce project scaladoc for compilation scope=macro.
scalor:scaladoc-main Yes Produce project scaladoc for compilation scope=main.
scalor:scaladoc-test Yes Produce project scaladoc for compilation scope=test.
scalor:sources-macro No Produce project sources for compilation scope=macro.
scalor:sources-main No Produce project sources for compilation scope=main.
scalor:sources-test No Produce project sources for compilation scope=test.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.5
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.carrotgarden.maven</groupId>
          <artifactId>scalor-maven-plugin_2.12</artifactId>
          <version>1.0.1.20180112035848</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.carrotgarden.maven</groupId>
        <artifactId>scalor-maven-plugin_2.12</artifactId>
        <version>1.0.1.20180112035848</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
  <!-- To use the report goals in your POM or parent POM -->
  <reporting>
    <plugins>
      <plugin>
        <groupId>com.carrotgarden.maven</groupId>
        <artifactId>scalor-maven-plugin_2.12</artifactId>
        <version>1.0.1.20180112035848</version>
      </plugin>
      ...
    </plugins>
  </reporting>
  ...
</project>
