scalor:eclipse-classpath

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.0.20171215014228:eclipse-classpath

Description:

Provision and configure Eclipse .classpath file.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
eclipseClasspathFile File - Location of Eclipse .classpath configuration file.
Default value is: ${project.basedir}/.classpath.
User property is: scalor.eclipseClasspathFile.
eclipseClasspathOrder String - Order Eclipse .classpath file entries according to these rules. Rule format: item = path, where: 'item' - relative sort order, 'path' - regular expression to match against 'classpath/classpathentry/@path',
Default value is: ; 01 = src/macro/java ; 02 = src/macro/scala ; 03 = src/macro/groovy ; 04 = src/macro/resources ; ; 11 = src/main/java ; 12 = src/main/scala ; 13 = src/main/groovy ; 14 = src/main/resources ; ; 21 = src/test/java ; 22 = src/test/scala ; 23 = src/test/groovy ; 24 = src/test/resources ; ; 31 = org[.]eclipse[.]jdt[.].+ ; 32 = org[.]scala-ide[.]sdt[.].+ ; 33 = org[.]eclipse[.]m2e[.].+ ; ; 41 = target/classes ; ; .
User property is: scalor.eclipseClasspathOrder.
eclipseClasspathProvide boolean - Provide default Eclipse .classpath file when missing.
Default value is: true.
User property is: scalor.eclipseClasspathProvide.
eclipseClasspathReorder boolean - Re-order Eclipse .classpath file by 'classpath/classpathentry/@path' .
Default value is: true.
User property is: scalor.eclipseClasspathReorder.
eclipseClasspathTemplate String - Scalor plugin classpath resource providing Eclipse .classpath file template. Used to provide project .classpath when missing.
Default value is: META-INF/template/.classpath.
User property is: scalor.eclipseClasspathTemplate.
eclipseOptionSeparator String - Eclipse settings option list separator.
Default value is: [;\n].
User property is: scalor.eclipseOptionSeparator.
skip boolean - Skip all steps of all executions.
Default value is: false.
User property is: scalor.skip.
